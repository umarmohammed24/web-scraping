# Web Scraper Pro

A full-stack Flask web application to scrape websites for title, text, images, and HTML source.

## Prerequisites

- Python 3.x installed on your system.

## Setup

1.  Open your terminal or command prompt.
2.  Navigate to the project directory:
    ```bash
    cd "c:/Users/UMAR MOHAMMED A/Desktop/WEB SCRAPINGG"
    ```
3.  Install the required dependencies:
    ```bash
    pip install flask requests beautifulsoup4
    ```

## Running the Application

1.  Start the Flask server:
    ```bash
    python app.py
    ```
2.  You will see output indicating the server is running (usually on port 5000).

## Usage

1.  Open your web browser and go to:  (http://127.0.0.1:5000)
2.  Enter a URL (e.g., `https://example.com`) in the input field.
3.  Click the **Scrape** button.
4.  View the extracted Title, Text, Images, and HTML Source.
